Status

Status@WIP:~$ |

This project is currently work in progress and not fully functional yet. Core ideas are in place, but training stability, game integration, and repeatable runs still need more work.

I left this page visible on purpose so the build process is transparent.

Current Scope

Scope@Prototype:~$ |

RL driving prototype for Trackmania input control

Early reward shaping based on speed + track progress

Game-state capture and action loop wiring

Basic logging to inspect training behavior

Experiment-friendly structure for iteration

Not production-ready and not consistently trainable yet

Planned Training Loop

Loop@Design:~$ |

1. Read frame or telemetry from Trackmania session

2. Infer steering/throttle action from policy network

3. Send action back to the game

4. Compute reward from speed, line quality, and checkpoint progress

5. Update policy and repeat

This loop exists conceptually but still needs reliability fixes
