Overview

Overview@City:~$ |

This project started as "can I generate a city that doesn't feel fake?" and turned into a traffic sandbox I kept rebuilding. Every run makes a new layout, then cars route through it so I can see where the network works and where it falls apart.

The point is less about making a pretty map and more about having a planning toy that reacts like a real city when the roads change.

What It Does

Features@Sandbox:~$ |

Generates a full city in passes: zones, parks, roads, cleanup, then buildings

Uses road classes (local, arterial, highway) so traffic has actual structure

Runs A* routing for cars and tracks congestion while the sim is live

Includes speed, pause, zoom, pan, and one-click "new city" regen

Shows useful HUD stats instead of just visuals: active cars, arrivals, stops, trip time

Has a separate Python side for experiments and environment-style testing

How It Is Built

Build@Modules:~$ |

I split this into two paths on purpose: browser for fast iteration, Python for cleaner simulation experiments.

city_visual.html is the interactive playground with rendering + controls.

grid.py and traffic_sim.py handle map structure, routing logic, occupancy checks, and metrics.

environment.py and visualize.py are the bridge toward testing and RL-style workflows.

How To Run It

Run@Quickstart:~$ |

1. Open city_visual.html and let it generate a fresh map.

2. Move around with zoom + pan and watch where jams form.

3. Hit New City a few times to compare different road topologies.

4. If you want Python mode, run main.py (visual/headless/envtest).

Version History

Version 7

Latest

External RL bridge and browser AI decoupling. Moved training out of the HTML runtime into Python modules, added a local bridge server for train/stop/status/apply control, and wired apply flow so the browser can send map state to Python and restore optimized layouts.

Version 6.5

Zoning visibility and district identity expansion. Added bottom-left zoning toggle with colorful zone-boundary outlines. Zone legend panel, organic zone shapes, and multiple special buildings per zone (landmarks, civic blocks, apartments, villas, plants) for improved visual variety and traffic hotspots.

Version 6.4

Traffic flow and density balancing pass. Fixed intersection deadlock behavior with relaxed entry gating while preserving collision safety. Reworked spawn behavior into demand-driven multi-spawn ticks with active-car cap of 500 for stable performance.

Version 6.3

Rendering and simulation performance merge. Optimized rendering pipeline with static offscreen caches. Replaced linear-scan A* with binary-heap priority queue. Removed per-tick full car sorting and added precomputed signal-intersection cache for faster traffic-light updates.

Version 6.2

Highway spacing stability pass. Updated highway generation to enforce dominant-axis corridor separation more reliably. Fixed near-parallel highway segments and improved motorway access control through better ramp point pruning.

Version 6.0

Editor and traffic AI upgrades. Added in-app city editor with paint tools for roads, zoning, and land edits. Exposed `window.cityEditor` API for scripted city changes. Merged stronger car recovery behavior and reworked road generation into a cleaner layered pipeline with increased city-shape variation.

Version 5.0

Full rework of browser simulation. Heavily refactored generation pipeline with layered approach. Highways, arterials, and local roads reorganized into realistic hierarchy. Building placement and city cleanup logic pushed further for more believable layouts.

Takeaways

Notes@After Build:~$ |

Cityflow is still in active development. Biggest lesson so far: drawing roads is easy, keeping a city network believable is the hard part. Most of the time went into connectivity, intersections, and cleaning up awkward generation outputs.

Next step is to unify the browser and Python logic so both modes behave the same.

Long-term direction: evolve Cityflow into a machine learning application where AI agents learn how to improve traffic flow and reduce congestion over time.
